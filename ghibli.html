The world of Studio Ghibli

Description

For this personal project, I wanted to create a webpage for one of my favorite animation studios: Studio Ghibli. One of the main challenges of this project was to use animations and CSS to make the different cards change size once the user hovers over one of them.

Since movies are the main inspiration, I wanted the experience of browsing through the site to immerse the user into the world that the amazing world that these movies have to offer. To obtain this result, I decided to have a short clip of the movie play before fading out to revealing the main article. I find this create a fun entrance and an immersive and cinematic experience, which was one of my goals for this project.
