Rootop greenhouse

Description

For this personal project, my goal was to create a communal rooftop garden. I was inspired by Concordia's own greenhouse and was interested in designing my own take on an open green space. After having sketched a rough idea on paper, I started to recreate the sketch in Blender. One of the challenges that I faced was learning how to add models such as plants and vines into Blender. In the end, I was able to recreate the vision I had when I first started this project and gain a better understanding on how to use Blender.

Thought process and inspiration

After having visited the Concordiaâ€™s rooftop green house for the first time, I was inspired to challenge myself and create a rooftop garden of my own on Blender. Elements that I wanted to add to my model were pergolas and picnic tables so that students could sit and study outside and have shade. I also decided to add a white roof for albedo effect and bright green accent walls to stay on theme. Since I wanted to create a green space, I wanted to be able to add plants and vines to my project. Thus, I taught myself how to use Blender block models and implement them in my personal project. By the end of this project, I was happy with the final result and was also happy to find myself being more comfortable using Blender as a new tool.
