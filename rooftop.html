Rootop greenhouse

Description

For this personal project, my goal was to create a communal rooftop garden. I was inspired by Concordia's own greenhouse and was interested in designing my own take on an open greenspace. After having stretched a rough idea on paper, I started to recreate the sketch in Blender. One of the challenges that I faced was learning how to add assets such as the plants and vines into Blender. In the end I was able to recreate the vision I had when I first started this project and gain a better understanding on how to use Blender.
